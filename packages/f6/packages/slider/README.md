### 介绍

`Slider` 用于滑动输入，常用于从某个范围中选择一个值。

### API

| 属性 | 描述 | 类型 | 默认值 |
| - | - | - | - |
| defaultValue | 当前数值 | number | `0` |
| min | 最小可选 | number | `0` |
| max | 最大可选 | number | `100` |
| runwayHeight | 轨道高度  | number | `2` |
| runwayColor | 轨道默认背景颜色 | string | - |
| trackColor | 轨道高亮背景颜色 | string | - |
| thumbColor | 手柄背景颜色 | string | - |
| step | 滑动倍数 | number | `1` |
| disabled | 禁用状态 | boolean | `false` |
