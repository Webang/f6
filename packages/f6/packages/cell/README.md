### 介绍

Cell 常用于列表中的展示项。

### Props

| 属性 | 说明 | 类型 | 默认值 |
| :-  | :- | :- | :- |
| title | 标题 | `ReactNode` | - |
| titleClass | 标题 className | `string` | - |
| label | 描述 | `ReactNode` | - |
| icon | 左侧图标 | `ReactNode` | - |
| value | 最右侧内容 | `ReactNode` | - |
| border | 是否显示底部边框 | `boolean` | `true` |
| onClick |	非必需，点击后触发的回调函数 | `React.MouseEventHandler<Element>` | - |
