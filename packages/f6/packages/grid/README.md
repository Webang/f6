### 介绍

`Grid` 格子布局，将容器分为多个单元格。

### GridProps

| 属性 | 说明 | 类型 | 默认值 |
| :-  | :- | :- | :- |
| border | 是否有边框 | `boolean` | `true` |
| columnNum | 每行展示的列数 | `number` | `4` |

### GridItemProps

| 属性 | 说明 | 类型 | 默认值 |
| :-  | :- | :- | :- |
| text | 展示的文字 | `ReactNode` | - |
| icon | 展示的图标 | `ReactNode` | - |
